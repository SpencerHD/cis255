MY FAVORITE HTML TAGS

Introduction:

This essay is to describe my three favorite tags for the Hypertext Markup Language (HTML). This document was made for CIS-255 SP18 at Saginaw Valley State University.

<abbr> The abbreviation tag is one of the first tags I learned about that I found very unique. With its ability to define an abbreviation or acronym, the user can easily hover over the text needing a defn., or explaining an acronym like HTML and be able to understand it without needing to write it out on the document. This can be especially useful when making a clean presented web document that saves space, and also looks more professional.

<select> The select tag is commonly used in webpages everywhere. Its that simple dropdown menu you use when you go to select the state you live in, for example, or the type of car you drive, i.e. . These can be useful when filling out forms, gathering information, or simply to give the user options when a certain answer is required. I like this tag not only because it can be very useful, but also because it is user friendly and fairly simple to implement.

<table> The table tag is extremely useful when presenting information in an organized fashion. With this tag, it allows you to use a number of columns and rows, as well as set a border for the table, then add the header and table contents as needed. You can modify the table in every way possible by using different styles, as well as combining columns and rows. This can be done for multiple tables in one document, setting the style for each one. An example of a table is used below, where the average temperatures for each season were calculated for Saginaw, Michigan.

Seasons in Saginaw Average Temperature

Spring 57.2 degrees F
Summer 68 degrees F
Fall 38.7 degrees F
Annual Average 47.6 degrees F
Conclusion:

While doing this assignment, I realized how many tags there are and how many purposes each one can be used for in a web document. The three I chose stood out to me as unique, while also useful in HTML to design a website. The abbreviation tag, select tag, and table tag seem to like they are very useful in this language, as well as others, which there are many to learned and apply for different situations.
